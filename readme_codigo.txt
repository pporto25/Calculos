# Cálculos :1234:

Aplicação C# para efetuar diversos tipos de cálculos

![Aplicação Cálculos](calculos.png)

Desenvolvida no *âmbito* da **_disciplina de TLP_**, **Introdução ao Software Git e à plataforma GitHub**

## Operações suportadas

Neste momento, esta aplicação implementa as seguintes operações:
- soma
- subtração
- conversão de temperaturas:
    - celsius:arrow_right: fahreneight
    - fahreneight:arrow_right: celsius

## Tecnologias usadas neste Projeto
- Visual Studio
- C#
- Git
- GitHub Desktop
- plataforma GitHub
